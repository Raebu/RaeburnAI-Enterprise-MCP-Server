# Roadmap

## v0.1 Hardened foundation

- Production TypeScript MCP stdio runtime
- Connector registry for enterprise tools
- Policy layer for allowlists, denylists and human approval
- Audit logging with redaction
- Docker, Docker Compose, CI, CodeQL and Dependabot
- Unit, integration and startup e2e tests

## v0.2 Admin setup and OAuth hardening

- OAuth device-code setup wizard
- Connector scope validation report
- GitHub App support alongside token auth
- Microsoft Graph permission inventory
- Slack app manifest example
- Salesforce Connected App setup guide

## v0.3 Enterprise approval flows

- Host-level approval callbacks
- Optional persisted approval queue
- Per-tool approval policy metadata
- Risk-based approval templates
- Admin audit export

## v0.4 Observability and hosted readiness

- OpenTelemetry metrics and traces
- Health and readiness HTTP sidecar option
- Tenant-aware configuration model
- Managed secret provider examples
- Connector sandbox contract tests

## v1.0 Stable enterprise release

- Backwards-compatible tool contracts
- Production deployment reference architecture
- Security review checklist complete
- Connector coverage expanded to Jira, Linear, ServiceNow and Zendesk
